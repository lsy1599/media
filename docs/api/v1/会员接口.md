# 会员接口

### 修改密码

接口：

```angular2html
/api/v1/member/password/change
```

请求参数：

| 字段 | 默认值 | 说明 |
| --- | --- | --- |
| `old_password` | | 原密码 |
| `new_password` | | 新密码 |
| `new_password_confirmation` | | 再输入一次新密码 |

返回参数：

> 成功返回空，下面是失败的信息。

```angular2html
{
    "message": "原密码不正确",
    "code": 500
}
```

### 我的课程

接口(GET)：

```angular2html
/api/v1/member/courses
```

请求参数：

| 参数名 | 默认值 | 说明 |
| --- | --- | --- | 
| `page` | `1` | 页码 |
| `page_size` | `10` | 每页显示条数 |

响应参数：

            "id": 1,

            "charge": 6,
            "short_description": "<p>Prof.</p>",

            "pivot": {
                "charge": 123,

| 字段 | 说明 |
| --- | --- |
| `data[].id` | 视频ID |

| `data[].short_description` | 简单介绍 |
| `data[].published_at` | 上线时间 |
| `data[].pivot.change` | 购买时的价格 |
| `data[].pivot.created_at` | 购买时间 |
